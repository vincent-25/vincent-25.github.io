汇编语言语句

汇编语言有两类语句

  • 执行性语句
  • 说明性语句
执行性语句
  • 表达处理器指令，实现功能
  • 标号: 硬指令助记符 操作数 ,操作数 ；注释
说明性语句
  • 表达伪指令，控制汇编方式
  • 名字 伪指令助记符 参数，参数， ；注释

区分标号与名字：标号后面有：，名字后面没有

  1. 标号指的是指令（处理器指令在主存中的逻辑地址）

    名字（自己定义的）指的是变量名，段名，子程序名等

  2. 助记符：帮助记忆指令功能的符号

    硬指令助记符表示处理器指令 （示例：MOV）

    伪指令助记符表示一个汇编命令

    （示例：字节变量定义 BYTE（或DB）msg byte ‘hello’,13,10,0）

  3. 数据部分

  4. 注释 以 ; 开头

8086汇编基本指令

从功能分类上来说，一共可分为：

一、 数据传送指令：MOV、XCHG、LEA、LDS、LES、PUSH、POP、PUSHF、POPF、CBW、CWD、CWDE。

二、 算术指令：ADD、ADC、INC、SUB、SBB、DEC、CMP、MUL、DIV、DAA、DAS、AAA、AAS。

三、 逻辑指令：AND、OR、XOR、NOT、TEST、SHL、SAL、SHR、SAR、RCL、RCR、ROL、ROR。

四、 控制转移指令：JMP、Jcc、JCXZ、LOOP、LOOPZ、LOOPNZ、LOOPNE、CALL、RET、INT。

五、 串操作指令：MOVS、LODS、STOS、CMPS、SCAS。

六、 标志处理指令：CLC、STC、CLD、STD。

七、 32位CPU新增指令

数据传送

MOV

  • 赋值指令

  • 写法：MOV target,source

  • 把源操作数source的值复制到target中去，source值不变

  • 注意：

    • target和source必须类型匹配

    • target不能是CS（代码段寄存器）

XCHG

  • 交换

  • 写法：XCHG object1,object2

  • 交换object1和2的值

LEA
  • 装入有效地址
  • LEZ REG,MEM
  • 将MEM的偏移地址送到REG中
LDS/LGS/LSS
  • 装入值
  • LDS reg16,mem32
  • 将mem指向的地址，高位放在DS中，低位存放到reg中
LES
  • LES reg,mem
  • 把mem指向的地址，高位存放在DS中，低位存放在reg中
PUSH
  • 入栈
  • PUSH SRC
  • 将SRC入栈
  • 注意：只能对字操作
POP
  • 出栈
  • POP DST
  • 将栈顶元素出栈赋值给DST
IN
  • IN DST,SRC
  • 从SRC端口读取一个字节/字到DST中
OUT
  • OUT DST,SRC
  • 将SRC中的数据送到DST短空中
XLAT
  • 取表指令
  • XLAT
  • 以DS:[BX+AL]为地址读取一个字节的数据送到AL中
PUSHF
  • PUSHF
  • 将PSW进栈
POPF
  • POPF
  • 将栈顶元素送到PSW
LAHF
  • LAHF
  • 将PSW的低八位送到AH中
SAHF
  • SAHF
  • 将AH写入PSW的低八位

算术

ADD
  • ADD DST,SRC
  • DST<-DST+SRC //DST=DST+SRC
ADC

  • 带进位加法指令

  • ADC DST,SRC

  • DST<-DST+SRC+CF

INC

  • 加1指令

  • INC DST

  • DST<-DST+1

SUB
  • 减法指令
  • SUB DST,SRC
  • DST<-DST-SRC
SBB

  • 带借位减法指令

  • SBB DST,SRC

  • DST<-DST-SRC-CF

DEC
  • 减1指令
  • DEC OPR
  • OPR<-OPR-1
NEG
  • 求补指令
  • NEG OPR
  • OPR<- -(OPR)
CMP
  • 比较指令
  • CMP OPR1,OPR2
  • OPR1-OPR2
  • 结果不保存，但是状态保存在PSW中，达到依据运算结果去影响EFLAG标志位的效果
MUL

  • 无符号数乘法

  • MUL SRC

  • 操作：

    字节操作：(AX)<-(AL)*(SRC)

    字操作：(DX,AX)<-(AX)*(SRC)

    双字操作：(EDX,EAX)<-(EAX)*(SRC)

IMUL
  • IMUL SRC
  • 有符号乘法
  • DST<-DST*SRC
DIV

  • 无符号数除法

  • 格式：DIV SRC

  • 功能：

    ①字节操作：(AH) <= (AX)/(SRC)的余数; (AL) <= (AX)/(SRC)的商

    ②字操作：(DX) <= (DX,AX)/(SRC)的余数; (AX) <= (DX,AX)/(SRC)的商

    ③双字操作：(EDX) <= (EDX,EAX)/(SRC)的余数; (EAX) <= (EDX,EAX)/(SRC)的商

IDIV

  • 无符号数除法

  • 格式：IDIV SRC

  • 功能：

    ①字节操作：(AH) <= (AX)/(SRC)的余数; (AL) <= (AX)/(SRC)的商

    ②字操作：(DX) <= (DX,AX)/(SRC)的余数; (AX) <= (DX,AX)/(SRC)的商

    ③双字操作：(EDX) <= (EDX,EAX)/(SRC)的余数; (EAX) <= (EDX,EAX)/(SRC)的商

CBW

  • 字节转化为字指令

  • CBW

  • 将AL保持原值（有符号数）扩展到AX

    即如果(AL)的最高有效位为0,则(AH)=00;如(AL)的最高有效位为1,则(AH)=0FFH

CWD
  • 字转化为双字指令
  • CWD
  • AX的内容符号扩展到DX.即如(AX)的最高有效位为0,则(DX)=0;否则(DX)=0FFFFH.

逻辑

AND

  • 与

  • AND DST,SRC

  • DST<-DST AND SRC

    将DST与SRC对应各位进行与运算后保留于DST

OR

  • OR DST,SRC

  • DST<-DST OR SRC

    将DST和SRC对应各位进行“或”运算后结果保留于DST

NOT
  • 描述：“非”逻辑运算
  • 格式： NOT DST
  • 功能：DST <- NOT DST 将DST各位”取反”结果保留于DST
OR

  • XOR DST,SRC

  • DST<-DST XOR SRC

    将DST和SRC对应各位进行“异或”运算后结果保留于DST

TEST
  • 与
  • TEST DST,SRC
  • 将DST和SRC各位进行与运算，结果状态影响PSW，但不影响DST,SRC的值
